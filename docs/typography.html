Typography

문자의 크기(Size), 행간(Line Height), 두께(Weight), 컬러(Color) 표현을 위한 스타일시트 입니다.

Title

스타일 가이드 Typography > Title1 ~ Title4 를 구현한 클래스 입니다.
기본 값의 line-height 는 스타일가이드 상의 Max 로 표현되며 *-tight 를 이용하여 Min 값으로 변경 가능 합니다.

Name Class Style
Title1 .ids2-title1 font-size: var(--ids2-size-title1); /* 32px */ line-height: var(--ids2-line-title1); /* 42px */
.ids2-title1-tight font-size: var(--ids2-size-title1); /* 32px */ line-height: var(--ids2-line-title1-tight); /* 40px */
Title2 .ids2-title2 font-size: var(--ids2-size-title2); /* 28px */ line-height: var(--ids2-line-title2); /* 40px */
.ids2-title2-tight font-size: var(--ids2-size-title2); /* 28px */ line-height: var(--ids2-line-title2-tight); /* 36px */
Title3 .ids2-title3 font-size: var(--ids2-size-title3); /* 26px */ line-height: var(--ids2-line-title3); /* 30px */
Title4 .ids2-title4 font-size: var(--ids2-size-title4); /* 24px */ line-height: var(--ids2-line-title4); /* 34px */
.ids2-title4-tight font-size: var(--ids2-size-title4); /* 24px */ line-height: var(--ids2-line-title4-tight); /* 32px */

Caption

스타일 가이드 Typography > Caption 를 구현한 클래스 입니다.
기본 값의 font-size 는 12px 로 표현되며 *-sm 를 이용하여 10px 로 변경 가능 합니다.
기본 값의 line-height 는 스타일가이드 상의 Max 로 표현되며 *-tight 를 이용하여 Min 값으로 변경 가능 합니다.

Name Class Style
Caption .ids2-caption font-size: var(--ids2-size-caption); /* 12px */ line-height: var(--ids2-line-caption); /* 16px */
.ids2-caption-tight font-size: var(--ids2-size-caption); /* 12px */ line-height: var(--ids2-line-caption-tight); /* 14px */
.ids2-caption-sm font-size: var(--ids2-caption-sm); /* 10px */ line-height: var(--ids2-line-caption-sm); /* 12px */

Font Weight

스타일 가이드 Typography > Font Weight(폰트 굵기) 를 구현한 클래스 입니다.
font-weight: 400 ~ 700 사이의 4가지 두께를 지원합니다.
기본적으로 Typography 스타일과 함께 사용 또는 단독 사용이 가능합니다.

Name Class Style
Font Weight .ids2-bold font-weight: var(--ids2-font-bold); /* 700 */
.ids2-semibold font-weight: var(--ids2-font-semibold); /* 600 */
.ids2-medium font-weight: var(--ids2-font-medium); /* 500 */
.ids2-regular font-weight: var(--ids2-font-regular); /* 400 */
