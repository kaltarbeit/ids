Icon

Icon 디자인 표현을 위한 스타일시트 입니다.

Size

스타일 가이드 Icon을 구현한 클래스입니다.
기본 24px(M) 을 베이스로 XXL, XL, L, S, XS, XXS 사이즈를 제공합니다.

Style Reference Table
Name Class Style
Base Icon (24px) .ids-icon display: inline-block; width: 24px; height: 24px; background-position: center; background-repeat: no-repeat; background-size: contain; font-size: 0; color: transparent; vertical-align: middle;
XXL Icon (36px) .ids-icon-xxl width: 36px; height: 36px;
XL Icon (32px) .ids-icon-xl width: 32px; height: 32px;
L Icon (28px) .ids-icon-lg width: 28px; height: 28px;
S Icon (18px) .ids-icon-lg width: 18px; height: 18px;
XS Icon (14px) .ids-icon-lg width: 14px; height: 14px;
XS Icon (12px) .ids-icon-lg width: 12px; height: 12px;

Color

Icon 의 Color 변경이 가능한 클래스 입니다.
아이콘 기본 SVG Fill Color #101116 에서 CSS Filter 를 통해 컬러를 변경합니다.
그러므로 추후 아이콘 추가/변경 시 SVG Fill Color #101116 로 등록해야 컬러 사용이 가능합니다.

Style Reference Table
Name Class Style
White .ids-icon-color-white filter: invert(1) brightness(3);
Gray 150 .ids-icon-gray-150 filter: invert(21%) sepia(85%) saturate(2398%) hue-rotate(199deg) brightness(94%) contrast(97%);
Primary .ids-icon-color-primary filter: invert(36%) sepia(87%) saturate(1582%) hue-rotate(196deg) brightness(95%) contrast(103%);
Point blue .ids-icon-color-point-blue filter: invert(21%) sepia(85%) saturate(2398%) hue-rotate(199deg) brightness(94%) contrast(97%);
Negative .ids-icon-negative filter: invert(18%) sepia(90%) saturate(5647%) hue-rotate(1deg) brightness(94%) contrast(106%);

Examples

.ids-icon .ids-icon-image .ids-icon-color-white
.ids-icon .ids-icon-image .ids-icon-gray-150
.ids-icon .ids-icon-image .ids-icon-color-primary
.ids-icon .ids-icon-image .ids-icon-color-point-blue
.ids-icon .ids-icon-image .ids-icon-color-negative
