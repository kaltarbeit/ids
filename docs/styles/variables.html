Line Height

IDS 에서 사용하는 Text Line Height 제공합니다.
Text Size에 대응하는 높이를 사용합니다.
Style Library > Typography > Type Scale 를 기반으로 작성되었습니다.

기본값은 Style Library > Typography > Type Scale의 각 스타일에 정의된 Line Height >Max에 대응하며, *-tight 는 Line Height > Min 에 대응합니다.

Font Weight

IDS 에서 사용하는 Font Weight를 제공합니다.
Style Library > Typography > Font Weight 를 기반으로 작성되었습니다.

Spacing

IDS 에서 사용하는 Margin / Padding / Gap Size를 제공합니다.
Style Library > Layout > Spacing 을 기반으로 작성되었습니다.

8px 배수를 기반으로 하며, *-half를 이용해 +0.5(4px) 표현이 가능합니다.
예) spacing-1 = 8px, spacing-2 = 16px, spacing-2-half = 20px, spacing-3 = 24px, ...

Border

IDS 에서 사용하는 Border를 제공합니다.
Style Library > Object > Border 를 기반으로 작성되었습니다.

*-[px]을 이용해 정의된 사이즈로 이용 가능합니다.
*-400, *-500를 이용해 미리 정의된 border-color 사용이 가능합니다.
*-hide를 이용해 border space 를 유지한 채로 border를 감출 수 있습니다.

Border Radius

IDS 에서 사용하는 Border Radius Size를 제공합니다.
Style Library > Object > Radius 를 기반으로 작성되었습니다.

*-[px]을 이용해 정의된 사이즈로 이용 가능합니다.
*-full을 이용해 반원 표현이 가능합니다.

Box Shadow

IDS 에서 사용하는 Box Shadow 스타일을 제공합니다.
Style Library > Elevation > Shadow 를 기반으로 작성되었습니다.
