Shadow

스타일 가이드 Elevation > Shadow 를 구현한 클래스입니다.

Name Class Style
Shadow .ids-shadow-1 box-shadow: var(--ids-shadow-1); /* 0 2px 4px 0 rgba(0, 0, 0, 0.06); */
.ids-shadow-2 box-shadow: var(--ids-shadow-2); /* 0 4px 6px 0 rgba(0, 0, 0, 0.08); */
.ids-shadow-3 box-shadow: var(--ids-shadow-3); /* 0 4px 8px 0 rgba(0, 0, 0, 0.1); */
.ids-shadow-4 box-shadow: var(--ids-shadow-4); /* 0 6px 20px 0 rgba(0, 0, 0, 0.14); */
.ids-shadow-5 box-shadow: var(--ids-shadow-5); /* 0 6px 14px 0 rgba(0, 0, 0, 0.24); */

Dimmed

스타일 가이드 Elevation > Dimmed 를 구현한 클래스입니다.
.ids-dimmed-hide 는 opacity 에 대한 Transition 만 제공할 뿐 실제 JavaScript 제어시 해당 요소를 display: none 처리를 수동으로 해주어야 합니다.

Style Reference Table
Name Class Style
Dimmed .ids-dimmed position: fixed; top: 0; left: 0; right: 0; bottom: 0; background-color: rgba(0, 0, 0, 0.5); z-index: 1000; transition: opacity 0.3s ease-in-out;
.ids-dimmed.ids-dimmed-hide opacity: 0;

Examples

.ids-dimmed
