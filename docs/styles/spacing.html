Spacing

Margin, Padding 표현을 위한 스타일시트 입니다.

Margin

스타일 가이드 Layout > Spacing을 기반으로 Margin 스타일을 구현한 클래스입니다.
8px 배수를 기반으로 하며, *-half를 이용해 +0.5(4px) 표현이 가능합니다.
예) ma-1 = 8px, ma-2 = 16px, ma-2-half = 20px, ma-3 = 24px, ...

Style Reference Table
Name Class Style
Margin All .ids-ma-1 margin: var(--ids-spacing); /* 8px */
.ids-ma-h margin: calc(var(--ids-spacing)/2); /* 4px */
.ids-ma-q margin: calc(var(--ids-spacing)/4); /* 2px */
.ids-ma-[number] margin: calc(var(--ids-spacing) * [number]);
.ids-ma-[number]-h margin: calc(var(--ids-spacing) * ([number] + 0.5));
Margin Top .ids-mt-[number] margin-top: calc(var(--ids-spacing) * [number]);
.ids-mt-[number]-h margin-top: calc(var(--ids-spacing) * ([number] + 0.5));
Margin Bottom .ids-mb-[number] margin-bottom: calc(var(--ids-spacing) * [number]);
.ids-mb-[number]-h margin-bottom: calc(var(--ids-spacing) * ([number] + 0.5));
Margin Left .ids-ml-[number] margin-left: calc(var(--ids-spacing) * [number]);
.ids-ml-[number]-h margin-left: calc(var(--ids-spacing) * ([number] + 0.5));
Margin Right .ids-mr-[number] margin-right: calc(var(--ids-spacing) * [number]);
.ids-mr-[number]-h margin-right: calc(var(--ids-spacing) * ([number] + 0.5));
Margin Vertical .ids-my-[number] margin-top: calc(var(--ids-spacing) * [number]); margin-bottom: calc(var(--ids-spacing) * [number]);
.ids-my-[number]-h margin-top: calc(var(--ids-spacing) * ([number] + 0.5)); margin-bottom: calc(var(--ids-spacing) * ([number] + 0.5));
Margin Horizontal .ids-mx-[number] margin-left: calc(var(--ids-spacing) * [number]); margin-right: calc(var(--ids-spacing) * [number]);
.ids-mx-[number]-h margin-left: calc(var(--ids-spacing) * ([number] + 0.5)); margin-right: calc(var(--ids-spacing) * ([number] + 0.5));
