Grid / 확장성 기반 그리드 시스템

Grid 표현을 위한 스타일시트와 동적 콘텐츠를 효과적으로 관리하기 위해 설계된 가이드입니다.

Columns

컴포넌트 스타일 가이드 Layout > 컬럼과 거터(Columns & Gutters), 여백(Margins) 를 구현한 클래스입니다.

기본 Margin (24px), Gutter (12px) 가 지정된 .ids-cols를 이용하여 4 column grid 사용이 가능하며, .ids-col-[number]로 1~4 까지의 column size 지정이 가능합니다.
.ids-cols-tight를 이용하면 Margin (20px), Gutter (8px) 로 변경 가능합니다.

Responsive Columns (확장성 기반 그리드 시스템)

컴포넌트 스타일 가이드 Layout > 반응형 레이아웃 그리드(Responsive Grid) 를 구현한 클래스입니다.

다양한 화면 크기와 해상도에 맞춰 콘텐츠를 효율적으로 배치할 수 있는 유연한 레이아웃 구조를 제공합니다.

.ids-col-lg-* 를 이용하여 8 column grid 지정 가능합니다.
.ids-col-lg-* 를 지정하지 않으면 Wide Screen에서도 기본 4 column grid 으로 동작합니다.
