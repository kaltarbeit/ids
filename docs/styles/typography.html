Typography

문자의 크기(Size), 행간(Line Height), 두께(Weight), 컬러(Color) 표현을 위한 스타일시트입니다.

Title

스타일 가이드 Typography > Title1 ~ Title4 를 구현한 클래스입니다.
기본 값의 line-height는 스타일가이드 상의 Max로 표현되며 *-tight 를 이용하여 Min값으로 변경 가능합니다.

Style Reference Table
Name Class Style
Title1 .ids-title1 font-size: var(--ids-size-title1); /* 32px */ line-height: var(--ids-line-title1); /* 42px */
.ids-title1-tight font-size: var(--ids-size-title1); /* 32px */ line-height: var(--ids-line-title1-tight); /* 40px */
Title2 .ids-title2 font-size: var(--ids-size-title2); /* 28px */ line-height: var(--ids-line-title2); /* 40px */
.ids-title2-tight font-size: var(--ids-size-title2); /* 28px */ line-height: var(--ids-line-title2-tight); /* 36px */
Title3 .ids-title3 font-size: var(--ids-size-title3); /* 26px */ line-height: var(--ids-line-title3); /* 30px */
Title4 .ids-title4 font-size: var(--ids-size-title4); /* 24px */ line-height: var(--ids-line-title4); /* 34px */
.ids-title4-tight font-size: var(--ids-size-title4); /* 24px */ line-height: var(--ids-line-title4-tight); /* 32px */

Caption

스타일 가이드 Typography > Caption 를 구현한 클래스입니다.
기본 값의 font-size 는 12px 로 표현되며 *-sm 를 이용하여 10px 로 변경 가능합니다.
기본 값의 line-height는 스타일가이드 상의 Max로 표현되며 *-tight 를 이용하여 Min값으로 변경 가능합니다.

Style Reference Table
Name Class Style
Caption .ids-caption font-size: var(--ids-size-caption); /* 12px */ line-height: var(--ids-line-caption); /* 16px */
.ids-caption-tight font-size: var(--ids-size-caption); /* 12px */ line-height: var(--ids-line-caption-tight); /* 14px */
.ids-caption-sm font-size: var(--ids-caption-sm); /* 10px */ line-height: var(--ids-line-caption-sm); /* 12px */

Examples

.ids-caption
.ids-caption-tight
.ids-caption-sm

Font Weight

스타일 가이드 Typography > Font Weight(폰트 굵기) 를 구현한 클래스입니다.
font-weight: 400 ~ 700 사이의 4가지 두께를 지원합니다.
기본적으로 Typography 스타일과 함께 사용 또는 단독 사용이 가능합니다.

Style Reference Table
Name Class Style
Font Weight .ids-bold font-weight: var(--ids-font-bold); /* 700 */
.ids-semibold font-weight: var(--ids-font-semibold); /* 600 */
.ids-medium font-weight: var(--ids-font-medium); /* 500 */
.ids-regular font-weight: var(--ids-font-regular); /* 400 */

Examples

.ids-bold
.ids-semibold
.ids-medium
.ids-regular
