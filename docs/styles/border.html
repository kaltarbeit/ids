Border

Border 디자인 표현을 위한 스타일시트 입니다.

Default

스타일 가이드 Object > Border 를 구현한 클래스입니다.
Gray Scale 100 으로 구현되어 있습니다.

Style Reference Table
Name Class Style
Border 1px .ids-border-1 border: var(--ids-border-1); /* 1px solid var(--ids-color-gray-100) */
Border 2px .ids-border-2 border: var(--ids-border-2); /* 2px solid var(--ids-color-gray-100) */
Border 3px .ids-border-3 border: var(--ids-border-3); /* 2px solid var(--ids-color-gray-100) */

Examples

.ids-border-1
.ids-border-2
.ids-border-3

Border Color

Default Border 에서 Border Color 변경을 위해 Gray Scale 150 ~ 900 과 white, primary, point-blue, negative 컬러를 함께 제공합니다.
ids-border-[px]-[color] 로 사용 가능합니다.
