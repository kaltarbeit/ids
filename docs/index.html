Overview

이 문서는 IBK Design System 을 바탕으로 작성된 Style Sheet 의 사용법 및 예제를 담고 있습니다.

좌측 메뉴를 통해 각 스타일 및 컴포넌트의 자세한 내용 및 예제를 보실 수 있습니다.

Prefix : ids-

IDS 의 모든 Class 명은 ids- 로 시작합니다.
이는 기존 CSS와 충돌을 막고 명확한 구분을 위한 방법입니다.

How to use

제공되는 CSS 파일은 아래와 같습니다.

  • styles
    • ids.min.css
    • icon.min.css
    • normalize.css
    • src
      • styles.css
      • variables.css
      • elevation.css
      • icon.css
      • grid.css
      • shape.css
      • skeleton.css
      • spacing.css
      • typography.css
      • components
        • accordion.css
        • badge.css
        • banner.css
        • ...

실 서비스에서의 사용은 ids.min.css를 이용합니다. 해당 파일은 src 하위 css의 minified 버전입니다.

커스터마이징 또는 소스 분석 시에는 src 하위 소스를 이용하면 됩니다.

IDS 2.0에서 제공하는 SVG Icon 사용을 원할 경우 icon.min.css를 함께 Import 해주면 됩니다.

만약 기본 Reset CSS가 없거나, 기존 CSS와 충돌이 날 경우 normalize.css를 이용해 보시기 바랍니다.
