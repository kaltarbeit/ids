Bottom Sheet

Bottom Sheet 형식의 디자인을 표현을 위한 스타일시트 입니다.

Default

컴포넌트 스타일 가이드 Banner > Small 를 구현한 클래스입니다. Base Style 과 조합으로 사용합니다.
해당 스타일은 예제 코드이므로 실제 사용시 각 디자인에 맞추어 구현해야 합니다.

Style Reference Table
Name Class Style
Default .ids2-bottom-sheet display: flex; position: absolute; bottom: 0; left: 0; right: 0; flex-direction: column; max-height: 80%; background-color: var(--ids2-color-white); border-top-left-radius: var(--ids2-radius-20); border-top-right-radius: var(--ids2-radius-20); transition: bottom 0.3s ease-in-out; z-index: 1001;
.ids2-bottom-sheet-header position: relative; padding: var(--ids2-spacing-3);
.ids2-bottom-sheet-content flex: 1 1 auto; overflow-y: auto; padding: var(--ids2-spacing-1) var(--ids2-spacing-2-half) 0;
.ids2-bottom-sheet-close position: absolute; top: 0; right: 0; padding: var(--ids2-spacing-3); display: flex; align-items: center; justify-content: center; background-color: transparent; border: transparent; cursor: pointer;

Examples

.ids2-bottom-sheet

With Button

컴포넌트 스타일 가이드 Bottom Sheet > With Button을 구현한 클래스입니다. Base Style과 조합으로 사용합니다.
Bottom Sheet 상단에 기능을 추가할 수 있는 .ids2-bottom-sheet-function와 하단에 .ids2-button-group을 추가하여 기능 확장이 가능합니다.

Style Reference Table
Name Class Style
Function .ids2-bottom-sheet-function position: absolute; top: -34px; padding: 0 var(--ids2-spacing-3);

Examples

.ids2-bottom-sheet

With Handle

컴포넌트 스타일 가이드 Bottom Sheet > With Handle을 구현한 클래스입니다. Base Style과 조합으로 사용합니다.
Bottom Sheet Header에 .ids2-bottom-sheet-handle를 추가하여 Touch Handler 영역을 표현할 수 있습니다.
