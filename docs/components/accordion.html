Accordion

Accordion 형식의 Collapse / Expand 구현을 위한 스타일시트 입니다.

Base Style

.ids-accordion 은 .ids-accordion-item 요소에 에 .ids-accordion-item-open 클래스를 Toggle 하여 Collapse / Expand 되도록 정의되어 있습니다.

.ids-accordion-item 은 제목과 클릭 요소를 담당하는 .ids-accordion-summary 와 Expand 시 표시 될 내용을 담는 .ids-accordion-details로 구성됩니다.
.ids-accordion-details 는 display: none 을 기본 값으로 하고, .ids-accordion-item-open 가 적용 될 경우 display: block으로 전환됩니다.

.ids-accordion-item-open .ids-accordion-arrow transform: rotate(-180deg);
.ids-accordion-item-open .ids-accordion-details display: block;
