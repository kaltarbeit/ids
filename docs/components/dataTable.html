Expandable

<tr> tag에 .ids2-row-expandable 를 적용하여 접힘/펼침 가능한 Row 기능을 사용 가능합니다.
.ids2-row-expandable 이 적용 되는 <tr> 에는 id 를 부여하고,
대상이 되는 .ids2-row-sub 이 적용 되는 <tr> 에는 data-parent-id 를 통해 부모와 동일한 id 를 부여하면 Grouping 가능합니다.
