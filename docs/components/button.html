Button

버튼 형식과 인터렉션 표현을 위한 스타일시트 입니다.

Base Button

<button>, <a> tag 의 기본 스타일을 Reset 하고, Button 의 Base Style 을 부여합니다.
단독으로는 사용할 수 없고, 아래 제공되는 스타일들과 조합으로 사용하거나 별도의 스타일을 구현하여 사용합니다.

Style Reference Table
Name Class Style
Base Button .ids2-button appearance: none; -webkit-font-smoothing: inherit; box-shadow: none; overflow:visible; cursor:pointer; display: flex; flex-direction: row; justify-content: center; align-items: center;

Primary Button

스타일 가이드 Button > Primary Buttons > Default의 스타일을 구현한 클래스 입니다.
Base Button > .ids2-button 클래스와 조합으로 사용됩니다.

Secondary Button

스타일 가이드 Button > Secondary Buttons > Default의 스타일을 구현한 클래스 입니다.
Base Button > .ids2-button 클래스와 조합으로 사용됩니다.
기본 높이 48px이며, .ids2-button-sm 클래스 조합을 통해 40px로 표현 가능합니다.

Tertiary Button

스타일 가이드 Button > Tertiary Buttons > Default의 스타일을 구현한 클래스 입니다.
Base Button > .ids2-button 클래스와 조합으로 사용됩니다.
기본 높이 40px이며, .ids2-button-sm 클래스 조합을 통해 32px로, .ids2-button-xs 클래스 조합을 통해 24px로 표현 가능합니다.
