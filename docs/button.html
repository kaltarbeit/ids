Button

의 버튼 형식과 인터렉션을 표현을 위한 스타일시트 입니다.

Base Button

<button>, <a> tag 의 기본 스타일을 Reset 하고, Button 의 Base Style 을 부여합니다.
단독으로는 사용할 수 없고, 아래 제공되는 스타일들과 조합으로 사용하거나 별도의 스타일을 구현하여 사용합니다.

Name Class Style
Base Button .ids2-button appearance: none; -webkit-font-smoothing: inherit; box-shadow: none; overflow:visible; cursor:pointer; display: flex; flex-direction: row; justify-content: center; align-items: center;

Primary Button

스타일 가이드 Button > Primary Buttons > Default의 스타일을 구현한 클래스 입니다.
Base Button > .ids2-button 클래스와 조합으로 사용됩니다.

Name Class Style
Primary .ids2-button-primary min-width: 100px; width: 100%; height: 56px; box-sizing: border-box; padding: var(--ids2-spacing-2); /* 16px */ gap: var(--ids2-spacing-1); /* 8px */ font-size: var(--ids2-size-h3); /* 18px */ font-weight: var(--ids2-font-bold); /* 700 */ line-height: var(--ids2-line-h3-tight); /* 24px */ color: var(--ids2-color-white); /* #FFFFFF */ background-color: var(--ids2-color-primary); /* #4393F0 */ border: var(--ids2-border-1-transparent); /* 1px solid transparent */ border-radius: var(--ids2-radius-12); /* 12px */
.ids2-button-primary:active background-color: var(--ids2-color-blue-900); /* #234EAA */
.ids2-button-primary:disabled font-size: var(--ids2-size-title1); /* 32px */ line-height: var(--ids2-line-title1-tight); /* 40px */

Examples

.ids2-button-primary
.ids2-button-primary:disabled

Secondary Button

스타일 가이드 Button > Secondary Buttons > Default의 스타일을 구현한 클래스 입니다.
Base Button > .ids2-button 클래스와 조합으로 사용됩니다.
기본 높이 48px이며, .ids2-button-sm 클래스 조합을 통해 40px로 표현 가능합니다.

Name Class Style
Secondary (Height 48) .ids2-button-secondary min-width: 100px; width: 100%; box-sizing: border-box; height: 48px; gap: var(--ids2-spacing-1); /* 8px */ border-radius: var(--ids2-radius-12); /* 12px */ font-size: var(--ids2-size-body2); /* 16px */ line-height: var(--ids2-line-body2-tight); /* 22px */ font-weight: var(--ids2-font-bold); /* 700 */ color: var(--ids2-color-primary); /* #4393F0 */ background-color: var(--ids2-color-white); /* #FFFFFF */ border: var(--ids2-border-1-primary); /* 1px solid #4393F0 */
.ids2-button-secondary:active background-color: var(--ids2-color-gray-100); /* #F5F5F5 */
.ids2-button-secondary:disabled color: var(--ids2-color-gray-500); /* #A1A1A1 */ background-color: var(--ids2-color-gray-200); /* #E5E5E5 */
Secondary (Height 40) .ids2-button-secondary.ids2-button-sm height: 40px;

Examples

.ids2-button-secondary
.ids2-button-secondary:disabled
.ids2-button-secondary .ids-button-sm

Tertiary Button

스타일 가이드 Button > Tertiary Buttons > Default의 스타일을 구현한 클래스 입니다.
Base Button > .ids2-button 클래스와 조합으로 사용됩니다.
기본 높이 40px이며, .ids2-button-sm 클래스 조합을 통해 32px로 표현 가능합니다.

Name Class Style
Tertiary .ids2-button-tertiary min-width: 100px; width: auto; height: 56px; box-sizing: border-box; gap: var(--ids2-spacing-1); /* 8px */ font-size: var(--ids2-size-h3); /* 18px */ font-weight: var(--ids2-font-bold); /* 700 */ line-height: var(--ids2-line-h3-tight); /* 24px */ color: var(--ids2-color-primary); /* #4393F0 */ background-color: transparent; border: none; border-radius: var(--ids2-radius-12); /* 12px */
.ids2-button-tertiary:active color: var(--ids2-color-blue-900); /* #234EAA */ background-color: var(--ids2-color-gray-100); /* #F5F5F5 */
.ids2-button-tertiary:disabled color: var(--ids2-color-gray-500); /* #A1A1A1 */ background-color: transparent;
